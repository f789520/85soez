## 使用技術

### 前端技術 <a name="前端技術"/>
* React 
    * React Router <a name="react-router"/>
      - SPA 路由功能 
    * React Context <a name="react-context"/>
      - 使用者狀態管理 
    * React Table : `useFilters`, `useSortBy`, `usePagination`...<a name="react-table"/>
      - 篩選 、 排序 、分頁功能
    * React Hooks : `useState`, `useEffect`, `useContext`...
* HTML
* CSS
* Javascript
* Responsive Web Design
* Webpack & Babel
* ESLint
* 第三方套件 : `emailjs`
 
### 後端技術 <a name="後端技術"/>
* Firebase(version 9)
    * Hosting <a name="hosting"/>
        - 網站上線
    * Authentication    <a name="authentication"/>
        - 會員註冊、一般登入、 Google 登入
    * Firestore  <a name="firestore"/>
        - 會員收藏清單資料儲存
    * Storage <a name="storage"/>
        - 會員大頭貼照片儲存
* Web Crawler
    * Python : `BeautifulSoup`
